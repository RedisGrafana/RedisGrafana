Redis Data Source 1.4.0

Redisgrafana.github.io

The initial release of documentation for Redis Data Source, Application and Explorer plugins published on https://redisgrafana.github.io.

Redis Application 1.1.0

The first Application plugin moved to React. Introduced RedisGears Script Editor, Latency and Keys consuming a lot of memory panels. 97% Unit tests coverage.

Redis Data Source 1.3.1

Added RedisGears, RedisGraph and Streams commands. 94% Unit tests coverage.

Redis 6.2

Redis 6.2 the "Community Edition" introduced new command, improvements, and bugfixes. Along with performance and memory optimizations.

Redis Data Source 1.3.0

Introduced Breaking Changes. Based on the latest version of Grafana 7.3. Added RediSearch commands.

Redis Data Source 1.2.1

Redis Labs Blogs

Series of blog posts "Introducing the Redis Data Source Plug-in for Grafana", "How to Use the New Redis Data Source for Grafana Plug-in" and "3 Real-Life Apps Built with Redis Data Source for Grafana".

Redis Data Source 1.2.0

Introduced new RedisTimeSeries commands, Streaming and ARM64 platform support for IoT.

Redis Data Source 1.1.2

The first community signed data source published in the Grafana repository. Based on the latest version of Grafana 7.1.

Itamar Haber did the first commit building a prototype for RedisTimeSeries using Radix client. Mikhail Volkov did another prototype which was merged to Redis Data Source.
