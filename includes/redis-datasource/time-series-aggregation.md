### Aggregation

| Type           | Description                                    |
| -------------- | ---------------------------------------------- |
| None (default) | No aggregation                                 |
| Avg            | Average                                        |
| Count          | Count number of samples                        |
| Max            | Maximum                                        |
| Min            | Minimum                                        |
| Range          | Diff between maximum and minimum in the bucket |
| Sum            | Summation                                      |
